## 新建帖子的 UI 组件 √

需要创建的组件，整体上是一个表单，因此根据需要提交的信息根据其 api 而定，如下：

- id - 是一个 UUID，使用随机数可以生成一个 24 位 16 进制数
- timestamp - 直接使用 Data.now() 生成
- title - 这是由用户输入的**表单输入框**
- body - 这是由用户输入的**表单输入框**
- author - 也就是作者，依然是**表单输入框**
- catagory - 所属的类别，是一个选择框，当然选择的方式可以是下拉或者其他。
